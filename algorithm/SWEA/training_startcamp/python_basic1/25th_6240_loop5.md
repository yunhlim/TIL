**문제 : 1부터 100사이의 숫자 중 3의 배수의 총합을 for 문을 이용해 출력하십시오.**

```
Input
없음



Output
1부터 100사이의 숫자 중 3의 배수의 총합: 1683
```
---
앞 두 문제랑 비슷한데 print가 더 쉬워서 간단히 clear~
```python
total = 0
for i in range(3,101,3):
    total += i
print(f'1부터 100사이의 숫자 중 3의 배수의 총합: {total}')
```
