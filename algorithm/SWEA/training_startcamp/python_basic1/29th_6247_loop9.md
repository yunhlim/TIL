print할 때 정렬!!

**f-strings**에서 가운데 정렬 하기 위해서 `{:^10}`을 사용. 10 자리 중 가운데 정렬!

복습해보면  

{:<} 왼쪽 정렬는  

{:>} 오른쪽 정렬 

{:.3} 정수 포함 총 3자리 

{:.3f} 소수점 3번째 자리까지

```python
star = '*'
cnt = 7
while True:
    if cnt > 0:
        print(f'{star*cnt:^7}')
    else:
        break
    cnt -= 2
```
