SWEA 6196. [파이썬 프로그래밍 기초(1) 파이썬의 기본 구조와 기초 문법] 4. 변수



**문제 : 1~9 사이의 정수 a를 입력받아 `a + aa + aaa + aaaa `의 값을 계산하는 프로그램을 작성하십시오.**

```
Input
9

Output
11106
```

***

입력을 어떻게 받는지 못찾아서 한참 헤맴..

input()으로 받으면 문자열이므로 int() 형변환 시켜서 받는다.

```python
a = int(input())
b = a*4 + a*10*3 + a*100*2+ a*1000
print(b)
```
