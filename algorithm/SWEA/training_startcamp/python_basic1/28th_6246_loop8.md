**문제 : while문을 이용해 아래와 같이 별(*)을 표시하는 프로그램을 만드십시오.**

```

Input

없음



Output

*****

****

***

**

*

```

---

파이썬에서 문자열끼리 합이나 곱은 지원하지만 차나 나눗셈은 지원하지 않는다.

```python

cnt = 0



while cnt < 5:

    star = 5-cnt

    cnt += 1

    print("*"*star)

```

결과 : **Pass**
